# ☆ Livraria Virtual - Android Java (AIDE)

Aplicativo Android desenvolvido em **Java**, criado no **AIDE**, simulando uma **livraria virtual** com sistema de login, cadastro e gerenciamento de livros.

Projeto desenvolvido com foco em **aprendizado prático de Java e desenvolvimento Android**, ideal para portfólio.

---

##  Funcionalidades

- Cadastro e login de usuários  
- Dois tipos de conta:
  -  **Leitor** → pesquisa e visualiza livros  
  - **Autor** → cadastra novos livros  
- Cadastro de livros com:
  - Título  
  - Nome do autor  
  - Preço  
  - Quantidade em estoque  
- Barra de pesquisa de livros  
- Banco de dados **SQLite** local  
- Interface simples e funcional  

---

## Tecnologias utilizadas

- Java  
- Android SDK  
- AIDE (Android IDE no celular)  
- SQLite  
- XML (layouts)

## Como executar no AIDE

1. Baixe o projeto (.zip)  
2. Extraia a pasta  
3. Abra o **AIDE**  
4. Clique em **Open Folder**  
5. Selecione a pasta do projeto  
6. Pressione **Run**
